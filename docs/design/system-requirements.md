
# システム要件

### 対応メディアソース
- バックエンドが稼働しているマシンのローカルディレクトリ
- SFTP接続先のディレクトリ
- AWS S3

### 対応メディア形式
- 画像ファイル: `png`, `jpg`, `jpeg`, `webp`, `gif`
- 動画ファイル: `mp4`, `webm`, `mov`, `mkv`, `avi` （yt-dlp/ffmpeg によるダウンロードとサムネイル生成）
- 音声ファイル: `mp3`, `wav`, `ogg`, `m4a`

### 機能要件

#### メディアソース管理
- ローカルディレクトリ、SFTP、AWS S3のメディアソースを登録、更新、削除できる。
- 各メディアソースへの接続をテストできる。
- メディアソースの現在の状態（例: 接続状況、スキャン状況）を取得できる。
- メディアソース内のディレクトリ構造を一覧表示できる。

#### メディア管理
- メディアの詳細情報（ファイルパス、ファイル名、タイプ、サイズ、日時など）を取得できる。
- メディアの評価、お気に入り、閲覧回数、最終閲覧日時を管理できる。
- AI生成メディアのメタデータ（プロンプト、ワークフロー、モデル名、シード値、CFGスケール、ステップ数など）を抽出・保存できる。
- メディアの技術情報（カラープロファイル、EXIFデータ、ハッシュ、動画/音声コーデックなど）を保存できる。
- メディアの同期ステータスとバックアップURLを管理できる。
- メディアのファイル名、説明、取得元リンク、タグを編集できる。
- メディアをアップロードできる（ファイル名指定、自動インクリメント、重複チェック、上書き許可）。
- メディアを検索・ソートできる（ファイル名、タグ、日付範囲、作成日時、更新日時、ファイルサイズ）。
    - タグ検索では、複数のタグを指定したAND検索（すべて含む）、OR検索（いずれかを含む）をサポートする。
    - 特定のタグを含まないメディアを検索するNOT検索（除外検索）をサポートする。

#### タグ管理
- タグのCRUD操作（作成、取得、更新、削除）ができる。
- タグの名前、説明、属性、色、起源（手動、AI生成など）を管理できる。
- メディアとタグの関連付けを管理し、AI生成タグの信頼度スコアを保存できる。

#### カテゴリ管理
- カテゴリのCRUD操作ができる。
- カテゴリ名、説明、色、階層構造を管理できる。

#### プロジェクト管理
- プロジェクトのCRUD操作ができる。
- プロジェクト名、説明、作成日時、アーカイブ日時を管理できる。

#### IP (知的財産) 管理
- IPのCRUD操作ができる。
- IP名、説明、起源を管理できる。
- メディアとIPの関連付けを管理し、AI生成IPの信頼度スコアを保存できる。

#### キャラクター管理
- キャラクターのCRUD操作ができる。
- キャラクター名、所属IP、説明、起源を管理できる。
- メディアとキャラクターの関連付けを管理し、AI生成キャラクターの信頼度スコアを保存できる。

#### ユーザー管理
- ユーザーのCRUD操作ができる。
- ユーザー名、メールアドレス、パスワードを管理できる。

#### コレクション管理
- コレクションのCRUD操作ができる。
- ユーザーに紐づくコレクション名、説明を管理できる。
- コレクションとメディアの関連付け、表示順序を管理できる。

#### 閲覧履歴
- メディアの閲覧日時、IPアドレス、ユーザーエージェントを記録できる。

#### 類似メディア
- 類似メディアの関連付け、類似度スコア、アルゴリズムを保存できる。

#### サムネイル管理
- メディアのサムネイルを配信できる。
- サムネイルの手動生成を開始できる。
- サムネイルキャッシュをクリアできる。

#### ディレクトリ管理
- ディレクトリの作成、リネーム、削除ができる。

#### リアルタイム更新
- ファイルシステムの変更（追加、削除、変更）やサムネイル生成の進捗状況をServer-Sent Events (SSE) でリアルタイムに通知できる。

#### 設定管理
- アプリケーション設定（サーバー設定、メディア処理設定、アップロード設定など）を取得、更新、デフォルトにリセットできる。
- 設定は`config.json`にJSON形式で保存され、更新時に自動バックアップされる。

#### ワークフロー・自動化機能
- サムネイル生成、メタデータ抽出、一括操作などのバックグラウンドジョブを管理できる。
- ジョブのステータス（待機中、実行中、完了、失敗）を追跡できる。

#### フィルタ・プリセット機能
- よく使う検索条件をプリセットとして保存し、再利用できる。

### 将来の機能要件 (雛形)

#### バルク操作機能
- 複数メディアの一括編集、削除、移動、タグ付け。

#### 統計・分析機能
- ソース統計、全体統計、重複メディア検出、類似メディア検索、人気メディアランキング。

#### エクスポート・インポート機能
- アーカイブエクスポート、インポート（URL/ファイル）、メディアダウンロード、ソース複製。

#### ワークフロー・自動化機能（拡張）
- ジョブのキューイング、優先度設定、キャンセル、再試行。
- AIによる自動タグ付けジョブの実行。
- 定期的なスキャンや同期の自動化。

#### 外部連携機能
- ComfyUIへの直接アップロード、ワークフロー一覧、Discord通知。
