# フロントエンド設計

このドキュメントでは、アプリケーションの主要なページ（画面）の構成と機能について記述します。

## ページ構成

### トップページ
- **ルーティング:** `/`
- **概要:** アプリケーションのデフォルトの初期画面です。現状では、SolidJSのテンプレートに含まれる "Hello World" メッセージが表示されます。将来的には、メディアソース一覧ページへのリダイレクトや、ダッシュボードのような機能が実装される可能性があります。

### メディアソース一覧ページ
- **ルーティング:** `/sources`
- **概要:** 登録されているメディアソースを管理する中心的な画面です。
- **実装済みの機能:**
  - 登録済みの全メディアソースをカード形式で一覧表示します。
  - 「Add Source」ボタンから、新しいメディアソースを追加するモーダルを開くことができます。
  - 各メディアソースカードから、既存のメディアソース情報を編集するモーダルを開くことができます。
  - 各メディアソースカードから、対象のメディアソースを削除する確認モーダルを開くことができます。
- **コンポーネント:**
  - `SourceCard`: メディアソースの情報を表示するカード。
  - `SourceFormModal`: メディアソースの追加・編集を行うフォームを持つモーダル。
  - `SourceDeleteModal`: メディアソースの削除を確認するモーダル。

### メディア一覧ページ
- **ルーティング:** `/sources/[mediaSourceId]`
- **概要:** 特定のメディアソースに含まれるメディアファイルを一覧表示する画面です。
- **実装済みの機能:**
  - メディアファイルのサムネイルをグリッドレイアウトで表示します。
  - サムネイルをクリックすると、メディア詳細ページに遷移します。
  - 画面全体がドラッグ＆ドロップのターゲットになっており、ファイルをドロップするとアップロードモーダルが開きます。
  - 画像やURLをページにペースト（貼り付け）すると、アップロードモーダルが開きます。
- **コンポーネント:**
  - `UploadMediaModal`: ファイル名や説明などを設定してメディアをアップロードするためのモーダル。
- **将来の構想:**
  - ファイル名、タグ、キャラクター、IPなどでメディアを絞り込む高度な検索バーの実装。
  - ディレクトリ階層を示すパンくずリストの表示。
  - ファイル名、作成日時、ファイルサイズなどでの並び替え機能。

### メディア検索ページ
- **ルーティング:** `/search`
- **概要:** システム全体でタグベースのメディア検索を行う専用画面です。
- **実装済みの機能:**
  - **メディアソース選択:** 検索対象のメディアソースを選択できます。
  - **タグ入力補完:** solid-uiの`Command`コンポーネントを使用し、システムに登録されているタグをオートコンプリートで選択できます。
  - **含むタグ選択:** 検索結果に含めたいタグを複数選択できます。選択されたタグはバッジとして表示され、削除可能です。
  - **除外タグ選択:** 検索結果から除外したいタグを複数選択できます。
  - **タグマッチモード:** AND（すべて含む）またはOR（いずれかを含む）を選択できます。
  - **ファイル名検索:** ファイル名による部分一致検索が可能です。
  - **ソート機能:** 作成日、ファイル名、ファイルサイズで昇順・降順にソートできます。
  - **ページネーション:** 検索結果のページング機能（次へ/前へボタン、結果数表示）。
  - **検索結果表示:** レスポンシブなグリッドレイアウトで検索結果を表示し、各メディアのサムネイル、ファイル名、パス、寸法、ファイルサイズを表示します。
- **使用API:**
  - `GET /api/tags` - タグ一覧取得（オートコンプリート用）
  - `GET /api/sources` - メディアソース一覧取得
  - `GET /api/sources/{mediaSourceId}/search` - メディア検索実行
- **技術スタック:**
  - solid-ui `Command`コンポーネント（タグ入力補完）
  - `clsx`と`tailwind-merge`（CSSクラスマージ）

### メディア詳細ページ
- **ルーティング:** `/sources/[mediaSourceId]/[mediaId]`
- **概要:** 一つのメディアファイルを大きく表示する画面です。
- **実装済みの機能:**
  - メディアファイル（現在は画像のみ）を画面中央に表示します。
- **将来の構想:**
  - ファイル名、解像度、タグ、カテゴリ、IP、キャラクターなどの詳細情報を表示するサイドパネルの実装。
  - 各種情報を直接編集できる機能。
  - AI生成メディアのプロンプトやワークフローを表示し、コピーや外部ツールへ転送する機能。
  - 関連メディア（バージョン違い、差分など）の表示。
