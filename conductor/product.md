# Initial Concept

solid-imager は、AI生成画像を含む大量のメディアファイルを、クリーンアーキテクチャに基づいて効率的に管理・検索・整理するためのWebアプリケーションおよびデスクトップクライアントです。

# Product Vision

大量のメディアファイル、特にAI生成画像におけるメタデータの価値を最大限に引き出し、ユーザーが「探す」ストレスから解放され、創作や鑑賞に集中できる環境を提供します。単なるファイルブラウザではなく、AI時代のメディア管理のスタンダードを目指します。

# Target Users

- **AI画像生成ユーザー**: Stable DiffusionやMidjourney等を頻繁に利用し、生成された画像とその背後にあるプロンプト情報を整理・管理したい個人。
- **クリエイター**: 素材収集や自身の作品管理において、高度なタグ付けやメタデータ検索を必要とする人々。
- **ガジェット愛好家**: NASやクラウドストレージ(S3等)に保存された大量のメディアを、ブラウザから透過的に、かつ高速に閲覧したいユーザー。

# Goals

- **情報の統合**: ローカル、SFTP、S3などの異なる物理ソースを意識させず、一つのライブラリとして統合管理する。
- **メタデータの再発見**: 画像に埋もれたプロンプトやモデル情報を自動抽出し、検索可能な資産に変える。
- **快適なブラウジング**: 数万枚規模の画像でも、高速なサムネイル表示とリアルタイムな同期により、滞りのない閲覧体験を実現する。

# Key Features (Priority)

1. **AIメタデータの自動抽出と高度な検索**: 生成情報のパースによるプロンプト検索や、AIを活用した類似画像検索・自動タグ付け。
2. **高速ブラウジングとリアルタイム同期**: サムネイル生成エンジンとSSE(Server-Sent Events)による、ファイル変更の即時反映。
3. **多角的なメタデータ管理**: 階層的なカテゴリ、タグ、キャラクター、作品(IP)といった、柔軟で強力な分類システム。
